## 1.distinct

####语法：

```
select distinct "字段名"
from "表格名"
```
####例子
在表格Store_Information中，找出所以不同的店名时。

```
select distinct store_name from Store_Information;
```
显示结果如下：

```
store_name
Los Angeles
San Diego
Bostan
```
